Emergency Access

Emergency access allows you to recover admin access when you've lost your password or can't log in normally. This feature uses your SECRET_KEY to generate a temporary access token that allows you to reset passwords or create new admin users.

How to use Emergency Access:

  1. Enable emergency access: export EMERGENCY_ACCESS_ENABLED=true
  2. Generate emergency token: uv run python scripts/emergency_access.py
  3. Visit: http://localhost:8000/oppman/emergency
  4. Enter your SECRET_KEY (not the token) in the form
  5. Reset your admin password or create a new superuser
  6. Disable emergency access: export EMERGENCY_ACCESS_ENABLED=false
